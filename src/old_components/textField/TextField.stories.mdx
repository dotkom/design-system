This component supports any props a regular `<input />` [accepts](https://www.w3schools.com/tags/tag_input.asp), as well as it's own props listed below.

## Status colors

These colors can be set by setting the `status` prop.  
Red color indicates a critical error, hindering the user in proceeding with what they are doing.  
Yellow color indicates that input _may_ be wrong, but is not validated.
This can also be used to alert users of fields that are usually faulty.
In this scenario, it is useful to set a label with a readable, but matching color.  
Green color indicates that input is correct and validated, and the user can submit, save or continue.
