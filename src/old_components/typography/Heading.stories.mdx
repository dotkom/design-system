# Headings

`<Heading>`, `<Heading2>`, `<Heading3>` components should be used instead of `<h1>`, `<h2>` and `<h3>`.

## Custom style

The component behaves just like a normal heading tag. Just use the `style` attribute if you need modifications.

  <Story name="Custom Style">
    <>
      <Heading style={{ color: 'red' }}>Custom styled Heading</Heading>
    </>
  </Story>
</Preview>
