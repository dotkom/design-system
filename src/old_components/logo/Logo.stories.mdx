# Compact Logo

This is the compact version of the logo, showing only the O with the lightning.
This can be used in places where the larger logo cannot fit, as a favicon, or application logo.
For rules regarding usage, see our [brand identity](https://online.ntnu.no/wiki/online/info/innsikt-og-interface/grafisk-profil/).
