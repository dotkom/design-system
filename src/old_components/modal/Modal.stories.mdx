# Modal

The modal is used to front important information, prompts, user-required actions and information hidden from default view.  
It accepts the props `open` and `onClose` which should be used to control whether the modal is displayed.  
`onClose` will be called by the modal on click of the exit button or if the user clicks outside the center component.

<Preview>
  <Story name="Modal">
    {() => {
      const [modalOpen, setModalOpen] = useState(false);
      return (
        <>
          <Modal open={modalOpen} onClose={() => setModalOpen(false)}>
            <p>This is the modal text</p>
          </Modal>
          <Button onClick={() => setModalOpen(true)}>Click me to display modal!</Button>
        </>
      );
    }}
  </Story>
</Preview>

# Modal with action

The modal will display any children it is passed.  
Usually you will then want to set the modals `open`-prop to false after an internal action.

<Preview>
  <Story name="Modal With Action Buttons">
    {() => {
      const [modalOpen, setModalOpen] = useState(false);
      return (
        <>
          <Modal open={modalOpen} onClose={() => setModalOpen(false)}>
            <h2 style={{ textAlign: 'center' }}>Accept terms?</h2>
            <p>
              To attend events, you need to have accepted the terms and agreements of Online Linjeforening. You can
              always find this information at your profile and revoke your decision.
            </p>
            <div style={{ display: 'flex', flexDirection: 'row', justifyContent: 'space-around', marginTop: '2em' }}>
              <Button onClick={() => setModalOpen(false)}>Accept Terms</Button>
              <Button variant="outline" onClick={() => setModalOpen(false)}>
                Do Not Accept
              </Button>
            </div>
          </Modal>
          <Button onClick={() => setModalOpen(true)}>Display modal with action buttons</Button>
        </>
      );
    }}
  </Story>
</Preview>
